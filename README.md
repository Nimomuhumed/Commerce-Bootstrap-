# MiniShop - Mini E-Commerce Website

This is a simple mini e-commerce website built with HTML, CSS, JavaScript, and Bootstrap.  
It includes a **cart system**, **user login/register**, and a **simulated M-Pesa payment** feature.

## Features
- Product listing
- Add to cart
- Checkout with M-Pesa simulation
- User authentication (login/register)
- GitHub Pages ready for live preview

## How to View
1. Upload all files to a GitHub repository
2. Enable **GitHub Pages** on the repository
3. Visit your GitHub Pages URL: `https://YOUR-USERNAME.github.io/ecommerce`

## Technologies Used
- HTML, CSS, JavaScript
- Bootstrap 5
- LocalStorage for cart & user authentication
